超好用的开源&免费软件推荐

发表于2022-04-04|更新于2022-04-04|pc
|阅读量:

如果软件在下载地址那下载速度过慢，推荐使用联想软件商店下载，下载地址：联想应用商店 (lenovo.com)

日常使用类

  • edge

    • 特点：免费
    • 介绍：微软出品的浏览器，使用体验非常棒，强烈推荐
    • 下载地址：下载 Microsoft Edge Web 浏览器 | Microsoft

  • 搜狗输入法

    • 特点：免费
    • 介绍：免费好用的输入法，词库强大，如果登录账号还可以同步词库，就是有广告，推荐使用火绒把广告屏蔽掉
    • 下载地址：搜狗输入法 - 首页 (sogou.com)

  • TIM

    • 特点：免费
    • 介绍：干净无花里胡哨版本的QQ
    • 下载地址：TIM (qq.com)

  • geek

    • 特点：免费
    • 介绍：简单好用的卸载软件
    • 下载地址：Geek Uninstaller - the best FREE uninstaller

  • bcuninstaller

    • 特点：免费 开源
    • 介绍：功能强大的卸载软件，但是有一定的入门门槛，推荐在b站看看教程在使用，如果不想看教程的推荐使用geek
    • 下载地址：Bulk Crap Uninstaller - Remove large amounts of unwanted applications (bcuninstaller.com)

  • 火绒

    • 特点：免费
    • 介绍：国产良心不流氓的杀毒软件，各项功能都堪称一绝
    • 下载地址：火绒安全 (huorong.cn)

  • 7zip

    • 特点：免费 开源
    • 介绍：简洁好用且开源的解压缩软件，就是界面和图标略有些古老
    • 下载地址：7-Zip

  • bandizip

    • 特点：免费 有广告
    • 介绍：功能和颜值都俱佳的解压缩软件，风靡全世界的解压软件，可惜新版免费版有广告，可以去百度找旧版使用，也可以直接用火绒屏蔽掉广告
    • 下载地址：Bandisoft - Bandizip, Honeycam, Honeyview

  • everything

    • 特点：免费
    • 介绍：秒搜软件，如果你找不到文件在哪了，又深受系统搜索速度的折磨，那么推荐使用这款软件
    • 下载地址：voidtools

  • coodesker

    • 特点：免费 开源
    • 介绍：免费开源的桌面整理软件
    • 下载地址：Coodesker - Desktop organize tool

      视频类

  • potplayer

    • 特点：免费
    • 介绍：堪称完美的本地视频播放器，功能强大
    • 下载地址：PotPlayer(网络播放器) v1.7.21620 绿色版 - 果核剥壳 (ghxi.com)

  • vlc

    • 特点：免费 开源
    • 介绍：免费开源的本地视频播放器，功能比肩potplayer，就是界面略显古老，不是颜值党的可以使用
    • 下载地址：VLC：官方网站 - 全平台的自由多媒体解决方案！ - VideoLAN

  • handbrake

    • 特点：免费 开源
    • 介绍：国内外非常知名的免费开源视频转码软件，功能非常强大，如果想做视频的可以下载一个
    • 下载地址：HandBrake: Open Source Video Transcoder

  • 剪映

    • 特点：免费
    • 介绍：国产超强的视频剪辑软件，功能强大易上手
    • 下载地址：剪映专业版-全能易用的桌面端剪辑软件-轻而易剪 上演大幕 (ulikecam.com)

  • kdenlive

    • 特点：免费 开源
    • 介绍：免费开源的视频剪辑软件，教程少，功能强大，但是存在一些bug，同时要求配置不低
    • 下载地址：https://kdenlive.org/zh/

  • shotcut

    • 特点：免费 开源
    • 介绍：免费开源的视频剪辑软件，功能基本够用，但是要求配置较高，且有一点的问题，但是上手难度较低，且有一点数量的教程
    • 下载地址：Shotcut - Home

  • DaVinci Resolve

    • 特点：免费
    • 介绍：免费版功能也十分的强大，但是配置要求极高
    • 下载地址：DaVinci Resolve 17 | Blackmagic Design

  • subtitle edit

    • 特点：免费 开源
    • 介绍：开源免费的视频字幕编辑软件
    • 下载地址：Nikse - Subtitle Edit

  • aegisub

    • 特点：免费 开源
    • 介绍：免费开源的视频字幕编辑软件
    • 下载地址：Aegisub 高级字幕编辑器 (vmoe.info)

  • arctime

    • 特点：免费
    • 介绍：国内使用数量庞大的视频字幕编辑软件
    • 下载地址：ArcTime Pro跨平台字幕软件 | Arctime字幕软件

  • quickcut

    • 特点：免费 开源
    • 介绍：超级好用的视频制作工具箱，包含转码等一系列功能
    • 下载地址：HaujetZhao/QuickCut: Your most handy video processing software (github.com)

  • 小丸工具箱

    • 特点：免费
    • 介绍：同上
    • 下载地址：小丸工具箱官方网站 (appinn.me)

      截图录像类

  • snipaste

    • 特点：免费 开源
    • 介绍：超级无敌好用的截图软件，用过就回不去了，建议作图的，搞编程的都可以下载一个
    • 下载地址：Snipaste - 截图 + 贴图

  • sharex

    • 特点：免费 开源
    • 介绍：被誉为截图界的瑞士军刀，功能无与伦比的强大，但是上手有一定的难度，但是b站有教程，推荐开源党可以使用
    • 下载地址：ShareX - The best free and open source screenshot tool for Windows (getsharex.com)

  • obs studio

    • 特点：免费 开源
    • 介绍：堪称世界最强的免费开源直播推流兼视频录制软件，功能极其强大，但是需要配置，有一定的上手难度，但是b站教程极多，不用担心学不会。
    • 下载地址：Open Broadcaster Software | OBS (obsproject.com)

  • captura

    • 特点：免费 开源
    • 介绍：简单好上手的视频录制软件，功能同样强大，且可录制gif，并且有鼠标点击高亮，键盘按键提示，推荐喜欢录制教程的朋友使用，可惜软件已经停止了维护
    • 下载地址：Captura (mathewsachin.github.io)

      设计类

  • Honeyview

    • 特点：免费
    • 介绍：堪称最顶级的免费看图软件，设计师必备，可以查看psd等图
    • 下载地址：Honeyview · 蜂蜜浏览器 · 免费的图像查看器 · 下载和功能说明 (bandisoft.com)

  • sagethumbs

    • 特点：免费 开源
    • 介绍：设计师必备软件，可以查看psd等图的略缩图，大大提升效率，体型小巧
    • 下载地址：SageThumbs - Download (softonic.com)

  • inkscape

    • 特点：免费 开源
    • 介绍：免费开源的矢量图绘制软件
    • 下载地址：Draw Freely | Inkscape

  • krita

    • 特点：免费 开源
    • 介绍：免费开源的绘画软件，功能极其强大，但是对于配置有一定的要求
    • 下载地址：www.krita.org

  • 图压

    • 特点：免费 开源
    • 介绍：免费开源的图片压缩软件
    • 下载地址：图压 - 简单易用的图片压缩软件 (xinxiao.tech)

      学习类

  • 思源笔记

    • 特点：免费 开源
    • 介绍：超级好用的本地笔记软件
    • 下载地址：思源笔记 - 本地优先的个人知识管理系统，支持 Markdown 排版、块级引用和双向链接 (b3log.org)

  • zotero

    • 特点：免费 开源
    • 介绍：超级无敌牛逼的开源文献管理软件，内置超级强大的pdf工具，推荐电脑学习党安装
    • 下载地址：Zotero | Your personal research assistant

  • calibre

    • 特点：免费 开源
    • 介绍：超级无敌牛逼的本地书籍管理软件
    • 下载地址：calibre - E-book management (calibre-ebook.com)

  • koodo

    • 特点：免费 开源
    • 介绍：颜值超级无敌高的免费开源电子书&漫画阅读器
    • 下载地址：Koodo Reader (960960.xyz)

  • sumatrapdf

    • 特点：免费 开源
    • 介绍：超级无敌轻量且开源的pdf查看器，打开pdf的速度超级快
    • 下载地址：Free PDF Reader - Sumatra PDF (sumatrapdfreader.org)

      编程类

  • notepad3

    • 特点：免费 开源
    • 介绍：超级好用的免费开源文本编辑器，用来看代码很舒服
    • 下载地址：Download Notepad3 - Notepad replacement with syntax highlighting. (rizonesoft.com)

  • vscode

    • 特点：免费 开源
    • 介绍：微软出品超强的文本编辑器，插件繁多，写前端必备
    • 下载地址：Visual Studio Code - Code Editing. Redefined

      效率类

  • utools

    • 特点：免费
    • 介绍：键盘党狂喜的超级效率软件
    • 下载地址：uTools官网 - 新一代效率工具平台

  • quicker

    • 特点：免费
    • 介绍：有免费版和收费版，个人感觉免费版够用，鼠标党狂喜的效率神器
    • 下载地址：Quicker软件 您的指尖工具箱 - Quicker (getquicker.net)

      折腾类

  • vmware player

    • 特点：免费
    • 介绍：vmware家出品的免费版虚拟机，功能强大
    • 下载地址：VMware Workstation Player | VMware

  • virtualbox

    • 特点：免费 开源
    • 介绍：免费开源的虚拟机软件，国内部分安卓模拟器的祖宗
    • 下载地址：Oracle VM VirtualBox

      远程控制类

  • todesk

    • 特点：免费
    • 介绍：免费的国产远程控制软件
    • 下载地址：ToDesk远程控制软件-免费安全流畅的远程连接电脑手机

  • rustdesk

    • 特点：免费 开源
    • 介绍：免费开源的远程控制软件
    • 下载地址：RustDesk | 开源远程桌面软件
